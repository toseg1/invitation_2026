# Quick Example: Using Your New Cards

## To See Your New Cards Immediately

Edit this ONE file: `src/config/invitation.config.ts`

### Change from this:

```typescript
layers: [
  'event-details',    // Bottom
  'crypto-stonks',    // Middle
  'code-triathlon',   // Top
]
```

### To this (to include your new cards):

```typescript
layers: [
  'event-details',    // Bottom - Final RSVP
  'samoa-island',     // NEW! - Samoa announcement
  'about-me',         // NEW! - Your photos & bio
  'crypto-stonks',    // Middle
  'code-triathlon',   // Top - Tear first
]
```

**Save the file and refresh your browser!** üöÄ

Now you have 5 layers:
1. **Tear** Code/Triathlon card (top)
2. **Tear** Crypto/Stonks card
3. **Tear** About Me card (your photos!)
4. **Tear** Samoa Island card (your announcement!)
5. **Reveal** Event Details (RSVP form)

---

## Recommended Order for Your Samoa Party

I suggest this storytelling order:

```typescript
layers: [
  'event-details',    // 5. Final: RSVP to the party
  'samoa-island',     // 4. Big reveal: Why you're leaving
  'about-me',         // 3. Who you are (with photos)
  'code-triathlon',   // 2. What you do (coding/sports)
  'crypto-stonks',    // 1. First tear: Fun intro
]
```

This tells the story:
1. Start with something fun (Crypto)
2. Show your hobbies (Code/Triathlon)
3. Introduce yourself with photos
4. Reveal the big news (Samoa!)
5. Invite them to celebrate
